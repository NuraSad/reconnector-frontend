## Setup
### Initial setup
1. Clone this repository.
2. Run `npm i` to install all dependancies.
3. Make a local .env file by copying the env.sample file.

### Setting up Supabase
1. Fill in `VITE_SUPABASE_URL` and `VITE_ANON_KEY` in your local .env file.

### Run the Application
```Run Application
npm run dev
```
